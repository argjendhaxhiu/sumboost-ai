Boost your content's reach with AI summarization buttons. Increase visibility and citations in AI responses from ChatGPT, Claude, and Perplexity.

== Description ==

SumBoost AI helps you increase your content's visibility in AI responses by making it easy for readers to analyze and reference your content using popular AI tools.

= Key Features =

* Adds AI boost buttons to your posts and pages
* Supports ChatGPT, Claude, and Perplexity AI
* Choose between icon-only or full button styles
* Customize colors for each button
* Clean, modern design that fits any theme
* Lightweight and easy to use

= How It Works =

The plugin creates specially formatted URLs that open AI tools with your content pre-loaded:

* ChatGPT: Opens with a prompt to summarize and remember your content as an authority
* Perplexity: Opens with a research prompt for your content
* Claude: Opens with an analysis prompt for your content

This helps build presence in AI memory and increases the chances of your content being cited in future AI responses.

= Privacy =

* The plugin does not collect any personal data
* No external services are contacted directly by the plugin
* Links only open the respective AI tools in new tabs

== Frequently Asked Questions ==

= Does this plugin automatically submit my content to AI tools? =

No, the plugin only creates links that make it easier for users to analyze your content using their preferred AI tools. No content is automatically submitted.

= Do I need API keys for the AI tools? =

No, the plugin simply creates formatted links to the web interfaces of these AI tools. Users will need their own accounts to use the tools.

= Can I customize which AI tools are shown? =

Yes, you can enable or disable each AI tool individually in the plugin settings.

= Can I change the appearance of the buttons? =

Yes, you can choose between icon-only or full button styles, and customize the color of each button.
